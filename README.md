      - Create Account ( Firebase )
      
      - Terms of Service -> Webview Terms of Service 
      - Privacy & Policy -> Webview Privacy & Policy
       
2. Home
3. Explore

4. Notification

      - Help & Feedback -> Webview Help & Feedback
